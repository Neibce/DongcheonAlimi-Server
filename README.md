## API 명세
### /schedules (시간표)
  - ```/:grade/:class```
### /exams (시험 일정)
  - ```/:year```
### /classes (학년 별 반 개수)
  - ```/count/:grade```
### /board (게시판)
  - ```/``` ```/new``` ```/list``` ```/:postId``` ```/:postId/delete``` ```/:postId/edit```
### /quiz (퀴즈)
  - ```/new``` ```/check-answer```
### /images (이미지 반환)
  - ```/:imageName```
